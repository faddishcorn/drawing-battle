# Google 로그인 활성화 방법

## Firebase Console에서 설정하기

1. **Firebase Console** 접속: https://console.firebase.google.com/

2. **프로젝트 선택** → drawing-battle-eaa0c

3. **Authentication** 클릭

4. **Sign-in method** 탭 클릭

5. **Google** 찾기 및 클릭

6. **활성화** 클릭 (토글 스위치)

7. **프로젝트 이름** 입력 (예: "배틀 드로잉")

8. **지원 이메일** 선택

9. **저장** 클릭

10. **배포 URI 추가** (개발 환경):
    - http://localhost:3000
    - https://localhost:3000

---

## 완료!

이제 앱에서 "Google로 로그인" 버튼을 클릭하면 Google 로그인 창이 나타날 것입니다.
